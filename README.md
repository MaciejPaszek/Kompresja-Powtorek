# Kompresja Powtórek  
## 1. Instalacja
### 1.1. Pobierz FFmpeg
Wejdź na https://github.com/BtbN/FFmpeg-Builds/releases i pobierz plik skompresowany o nazwie `ffmpeg-master-latest-win64-gpl-shared.zip`. Rozpakuj archiwum i jego zawartość umieść w dowolnej lokalizacji na dysku, n.p. `C:\FFmpeg`. 

<img width="643" height="368" alt="image" src="https://github.com/user-attachments/assets/9e7ef8a7-a37f-4906-8f4c-11ae82b676bc" />

### 1.2. Dodaj FFmpeg do zmiennej środowiskowej PATH
Wejdź w `Ustawienia > System > Informacje > Zaawansowane Ustawienia Systemu > Zmienne Środowiskowe`.
Dwukrotnie kliknij w zmienną Path, lub zaznacz ją i wybierz Edytuj.
W oknie edycji kliknij Nowy i podaj ścieżkę do wolderu, w którym znajdują się pliki `FFmpeg.exe`, n.p. `C:\FFmpeg\bin` 

<img width="1612" height="612" alt="image" src="https://github.com/user-attachments/assets/59cc369a-96d1-4742-b632-8edcc8895f74" />

### 1.3 Pobierz program Kompresja Powtórek z sekcji Releases.
Pobierz plik `Kompresja.Powtorek.zip`, rozpakuj go i uruchom program `Kompresja Powtóek.exe`.
